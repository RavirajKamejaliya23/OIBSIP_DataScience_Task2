# OIBSIP_DataScience_Task2

## Unemployment Analysis with Python

### Objective

The objective of this project is to analyze unemployment trends in India and understand the impact of various factors, including the Covid-19 pandemic, on unemployment rates.

### Dataset

The dataset contains information about:

* State
* Date
* Frequency
* Unemployment Rate
* Employed Population
* Labour Participation Rate
* Region

### Tools and Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

### Steps Performed

1. Loaded and explored the dataset.
2. Cleaned and preprocessed the data.
3. Analyzed unemployment rates across different states and regions.
4. Created visualizations including histograms, boxplots, scatter plots, and heatmaps.
5. Studied monthly unemployment trends.
6. Evaluated the impact of Covid-19 on employment patterns.

### Result

Successfully analyzed unemployment trends and identified key patterns across different regions of India.

### Outcome

This project enhanced understanding of data analysis, visualization, and extracting meaningful insights from real-world datasets.
